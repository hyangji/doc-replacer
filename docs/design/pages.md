# DocReplacer - 주요 화면 와이어프레임

## 화면 1: 메인 대시보드 (`/`)

### 와이어프레임

```
+------------------------------------------------------------------+
|  [Breadcrumb: 홈 > 대시보드]                                      |
+------------------------------------------------------------------+
|                                                                    |
|  +--------------------------------------------------------------+ |
|  |                  파일 업로드 영역                               | |
|  |                                                                | |
|  |     +--------------------------------------------------+      | |
|  |     |                                                  |      | |
|  |     |    [CloudUploadOutlined 아이콘 (48px)]           |      | |
|  |     |                                                  |      | |
|  |     |    HWP, HWPX, 엑셀 파일을 드래그하거나           |      | |
|  |     |    클릭하여 업로드하세요                           |      | |
|  |     |                                                  |      | |
|  |     |    [파일 선택 버튼]                               |      | |
|  |     +--------------------------------------------------+      | |
|  |           지원 형식: .hwp, .hwpx, .xlsx  |  최대 50MB          | |
|  +--------------------------------------------------------------+ |
|                                                                    |
|  +--- 퀵 액션 ---------------------------------------------------+ |
|  |  [+ 새 문서 작성]  [파일 변환]  [엑셀 일괄 수정]  [법률 검색]  | |
|  +--------------------------------------------------------------+ |
|                                                                    |
|  +--- 최근 문서 -------------------- [전체 보기 >] --------------+ |
|  |                                                                | |
|  | 파일명           | 유형   | 수정일       | 상태    | 액션      | |
|  |-----------------|--------|-------------|---------|-----------|  |
|  | 제안서_v3.hwpx  | HWPX   | 2026-04-25  | 수정됨  | [편집][삭제]|
|  | 계약서_초안.hwp  | HWP    | 2026-04-24  | 원본   | [편집][삭제]|
|  | 수정데이터.xlsx  | Excel  | 2026-04-23  | 매핑됨  | [보기][삭제]|
|  | 도시계획_v2.hwpx | HWPX   | 2026-04-22  | 변환됨  | [편집][삭제]|
|  | ...              |        |             |         |           | |
|  |                                                                | |
|  |                    [1] [2] [3] ... [페이지네이션]               | |
|  +--------------------------------------------------------------+ |
+------------------------------------------------------------------+
```

### 사용 컴포넌트

| 영역 | Ant Design 컴포넌트 | 설명 |
|------|---------------------|------|
| 파일 업로드 | `Upload.Dragger` | 드래그앤드롭 업로드 영역 |
| 퀵 액션 | `Space` + `Button` (type="default", icon 포함) | 주요 기능 바로가기 |
| 최근 문서 테이블 | `Table` | 정렬, 필터, 페이지네이션 포함 |
| 상태 태그 | `Tag` (color별 구분) | 원본=blue, 수정됨=orange, 변환됨=green |
| 액션 버튼 | `Button` (type="link") + `Popconfirm` (삭제 시) | 행 단위 액션 |
| 섹션 제목 | `Typography.Title` (level=4) | 각 섹션 헤더 |

### 상호작용

- 파일 드롭 시 자동으로 파일 유형 감지 후 업로드 시작 (`Progress` 표시)
- 업로드 완료 후 문서 유형에 따라 편집기 또는 매핑 화면으로 자동 이동
- 테이블 행 클릭 시 해당 문서 편집기로 이동
- 퀵 액션 버튼은 각각 해당 화면으로 라우팅

## 화면 2: 문서 편집기 (`/editor/:documentId`)

### 와이어프레임

```
+------------------------------------------------------------------+
|  [툴바]                                                           |
|  [저장 💾] [되돌리기 ↩] [다시하기 ↪] | [변환 ▼] | [Diff] [전체화면]|
+------------------------------------------------------------------+
|                                           | 검색/치환 패널 (토글) |
|                                           | [x 닫기]             |
|                                           +----------------------|
|                                           | 찾기:                |
|   Monaco Editor                           | [입력필드    ] [Aa]   |
|                                           | [.*] [↑] [↓]         |
|   1 | 제1조(목적)                          | 3개 결과 중 1번째     |
|   2 | 이 조례는 도시계획 제안서의...         |----------------------|
|   3 |                                      | 바꾸기:              |
|   4 | 제2조(정의)                           | [입력필드    ]        |
|   5 | 이 조례에서 사용하는 용어의 ...        | [바꾸기] [모두 바꾸기] |
|   6 |                                      +----------------------+
|   7 | 1. "도시계획"이란 ...                  |
|   8 |   가. 도시기본계획                     |
|   9 |   나. 도시관리계획                     |
|  10 |                                       |
|  11 | 제3조(적용범위)                        |
|  12 | ...                                   |
|                                           |
+------------------------------------------------------------------+
| [상태바] 줄: 1, 열: 1 | UTF-8 | 총 245줄 | 검색: 3개 일치        |
+------------------------------------------------------------------+
```

### 검색/치환 패널 상세

```
+------------------------------------------+
| 검색/치환                         [x 닫기]|
+------------------------------------------+
| 찾기:                                     |
| +----------------------------+ [Aa] [.*]  |
| | 검색어 입력                 |  [↑] [↓]  |
| +----------------------------+            |
| 3개 결과 중 1번째                          |
+------------------------------------------+
| 바꾸기:               (펼침/접힘 토글)     |
| +----------------------------+            |
| | 바꿀 단어 입력              |            |
| +----------------------------+            |
| [하나씩 바꾸기] [모두 바꾸기]              |
+------------------------------------------+
```

### 사용 컴포넌트

| 영역 | Ant Design / 라이브러리 | 설명 |
|------|------------------------|------|
| 툴바 | `Space` + `Button` + `Divider` (vertical) + `Dropdown` | 편집 도구 모음 |
| 에디터 | `Monaco Editor` (@monaco-editor/react) | 코드/문서 편집기 |
| 검색/치환 패널 | 커스텀 컴포넌트 (`Input`, `Button`, `Tooltip`) | IDE 스타일 검색 |
| Aa 버튼 | `Button` (size="small") + `Tooltip` ("대소문자 구분") | 대소문자 토글 |
| .* 버튼 | `Button` (size="small") + `Tooltip` ("정규식") | 정규식 토글 |
| 위/아래 화살표 | `Button` (icon=`UpOutlined`/`DownOutlined`) | 검색 결과 탐색 |
| 상태바 | 커스텀 `div` (Flex 레이아웃) | 하단 정보 표시 |
| 변환 드롭다운 | `Dropdown` + `Menu` | Word/PDF 변환 옵션 |

### 상호작용

- `Ctrl+F`: 검색 패널 열기 (찾기만)
- `Ctrl+H`: 검색/치환 패널 열기 (바꾸기 포함)
- `Escape`: 검색 패널 닫기
- `Enter`: 다음 검색 결과로 이동
- `Shift+Enter`: 이전 검색 결과로 이동
- 검색 결과는 에디터 내에서 하이라이트 표시 (노란색 배경)
- 현재 선택된 결과는 주황색 배경으로 구분
- `Ctrl+S`: 저장 (API 호출)
- `Ctrl+Z` / `Ctrl+Shift+Z`: 되돌리기 / 다시하기

## 화면 3: Diff View (`/diff/:documentId`)

### 와이어프레임

```
+------------------------------------------------------------------+
|  [Breadcrumb: 홈 > Diff 보기 > 제안서_v3.hwpx]                   |
+------------------------------------------------------------------+
|  [툴바]                                                           |
|  보기: (o)나란히 ( )통합  |  [이전변경] [다음변경]  | [되돌리기]   |
+------------------------------------------------------------------+
|          원본 문서                |        수정된 문서             |
|----------------------------------|--------------------------------|
|                                  |                                |
|  1 | 제1조(목적)                  |  1 | 제1조(목적)               |
|  2 | 이 조례는 ███████ 제안서의   |  2 | 이 조례는 ███████ 제안서의|
|    | (빨간 배경 - 삭제)           |    | (초록 배경 - 추가)        |
|  3 |                              |  3 |                           |
|  4 | 제2조(정의)                  |  4 | 제2조(정의)               |
|  5 | 이 조례에서 사용하는 용어의  |  5 | 이 조례에서 사용하는 용어의|
|  6 |   "███"이란 (빨간 배경)      |  6 |   "███"이란 (초록 배경)   |
|  7 |                              |  7 |                           |
|  ...                             |  ...                          |
|                                  |                                |
+------------------------------------------------------------------+
|  [변경사항 요약]                                                   |
|  +--------------------------------------------------------------+ |
|  | #  | 위치    | 유형 | 원본 내용        | 수정 내용       | 액션  ||
|  |----|--------|------|-----------------|----------------|-------||
|  | 1  | 2줄    | 수정 | "도시계획"      | "도시개발"      | [되돌리기]|
|  | 2  | 6줄    | 수정 | "정비사업"      | "재개발사업"    | [되돌리기]|
|  | 3  | 15줄   | 삭제 | "제5항 삭제"    | -              | [되돌리기]|
|  | 4  | 22줄   | 추가 | -               | "신규 조항"    | [되돌리기]|
|  +--------------------------------------------------------------+ |
|                                                                    |
|  [모두 되돌리기]  [선택 되돌리기]  [저장]  [Word로 내보내기]        |
+------------------------------------------------------------------+
```

### 사용 컴포넌트

| 영역 | Ant Design / 라이브러리 | 설명 |
|------|------------------------|------|
| Diff 뷰어 | `react-diff-viewer` (ReactDiffViewer) | 나란히/통합 모드 지원 |
| 보기 모드 전환 | `Radio.Group` | 나란히(split) / 통합(unified) |
| 이전/다음 변경 | `Button` (icon=`UpOutlined`/`DownOutlined`) | 변경 지점 탐색 |
| 변경사항 테이블 | `Table` | 변경 목록, 행 선택 가능 |
| 유형 태그 | `Tag` | 수정=orange, 삭제=red, 추가=green |
| 되돌리기 버튼 | `Button` (danger) + `Popconfirm` | 개별 변경 되돌리기 |
| 하단 액션 | `Space` + `Button` | 일괄 액션 버튼 그룹 |

### 상호작용

- 변경사항 테이블 행 클릭 시 해당 Diff 위치로 스크롤
- 개별 되돌리기: 해당 변경만 원본으로 복원
- 선택 되돌리기: 체크박스로 선택한 변경들만 복원
- Diff 뷰어 내에서 변경된 영역 호버 시 툴팁으로 변경 상세 표시
- 나란히/통합 모드 전환 시 현재 스크롤 위치 유지

## 화면 4: 파일 업로드/변환 (`/files/upload`)

### 와이어프레임

```
+------------------------------------------------------------------+
|  [Breadcrumb: 홈 > 파일 관리 > 업로드/변환]                       |
+------------------------------------------------------------------+
|                                                                    |
|  +--- Step 1: 파일 업로드 ---- Step 2: 매핑 ---- Step 3: 결과 ---+|
|  |  (●)                      ( )                  ( )             ||
|  +--------------------------------------------------------------+ |
|                                                                    |
|  +--- 문서 파일 업로드 --------+  +--- 수정 데이터 업로드 --------+|
|  |                             |  |                               ||
|  |  [Upload.Dragger]           |  |  [Upload.Dragger]             ||
|  |                             |  |                               ||
|  |  HWP / HWPX 파일           |  |  엑셀 파일 (.xlsx)            ||
|  |  을 업로드하세요             |  |  수정 데이터를 업로드하세요    ||
|  |                             |  |                               ||
|  |  [파일목록]                 |  |  [파일목록]                   ||
|  |  - 제안서_v3.hwpx  [x삭제] |  |  - 수정사항.xlsx  [x삭제]    ||
|  |                             |  |                               ||
|  +-----------------------------+  +-------------------------------+|
|                                                                    |
|  +--- 파일 변환 옵션 --------------------------------------------+|
|  |                                                                ||
|  |  출력 형식:  [x] Word (.docx)  [ ] PDF (.pdf)                 ||
|  |                                                                ||
|  |  변환 옵션:                                                    ||
|  |  [x] 서식 유지    [ ] 이미지 포함    [x] 목차 생성            ||
|  |                                                                ||
|  +--------------------------------------------------------------+ |
|                                                                    |
|  +--- 매핑 미리보기 (엑셀 데이터 기반) --------------------------+|
|  |                                                                ||
|  | 원본 텍스트          | 변경할 텍스트        | 위치   | 상태    ||
|  |---------------------|---------------------|--------|---------|  |
|  | "도시계획"           | "도시개발"           | 전체   | 대기    ||
|  | "2025년"            | "2026년"             | 전체   | 대기    ||
|  | "정비사업"           | "재개발사업"          | 3페이지 | 대기   ||
|  | "1,000,000원"       | "1,500,000원"        | 5페이지 | 대기   ||
|  |                                                                ||
|  +--------------------------------------------------------------+ |
|                                                                    |
|  [취소]                           [미리보기]  [변환 실행]          |
+------------------------------------------------------------------+
```

### 사용 컴포넌트

| 영역 | Ant Design 컴포넌트 | 설명 |
|------|---------------------|------|
| 단계 표시 | `Steps` | 3단계 진행 표시 (업로드 > 매핑 > 결과) |
| 파일 업로드 | `Upload.Dragger` (2개) | 문서 파일 / 엑셀 파일 별도 업로드 |
| 파일 목록 | `Upload`의 `fileList` | 업로드된 파일 목록 표시 |
| 변환 옵션 | `Checkbox.Group` | 출력 형식 및 변환 옵션 선택 |
| 매핑 테이블 | `Table` (editable cells) | 셀 클릭하여 수정 가능 |
| 상태 태그 | `Tag` | 대기=default, 성공=green, 실패=red |
| 하단 액션 | `Space` + `Button` (primary="변환 실행") | 실행 / 취소 |
| 진행률 | `Progress` + `Modal` | 변환 중 진행률 표시 |

### 상호작용

- 엑셀 업로드 시 자동으로 매핑 테이블 생성 (엑셀의 원본/변경 컬럼 파싱)
- 매핑 테이블에서 직접 수정 가능 (인라인 편집)
- "미리보기" 클릭 시 변환 결과를 Diff View로 표시 (모달)
- "변환 실행" 클릭 시 진행률 모달 표시 후 완료 시 결과 화면으로 이동
- Step 진행에 따라 이전/다음 버튼으로 단계 이동

## 화면 5: 법률 검색 (`/law-search`)

### 와이어프레임

```
+------------------------------------------------------------------+
|  [Breadcrumb: 홈 > 법률 검색]                                     |
+------------------------------------------------------------------+
|                                                                    |
|  +--- 검색 영역 ------------------------------------------------+|
|  |                                                                ||
|  |  검색 유형: (o)법령명  ( )조문번호  ( )키워드                  ||
|  |                                                                ||
|  |  +--------------------------------------------------+ [검색]  ||
|  |  | 도시계획법                                        |         ||
|  |  +--------------------------------------------------+         ||
|  |                                                                ||
|  |  [고급 검색 ▼]                                                 ||
|  |  ┌─ 고급 검색 옵션 (접힘/펼침) ─────────────────────┐         ||
|  |  │ 법령 구분: [x]법률 [x]시행령 [x]시행규칙 [ ]조례 │         ||
|  |  │ 시행일: [2020-01-01] ~ [2026-12-31]              │         ||

|  |  └──────────────────────────────────────────────────┘         ||

|  |  국토의 계획 및 이용에 관한  |  |  제2조(정의)                 ||
|  |  법률                        |  |                              ||
|  |  법률 제19XXX호 | 시행 2026. |  |  이 법에서 사용하는 용어의   ||

|  |  도시개발법                  |  |                              ||
|  |  법률 제19XXX호 | 시행 2025. |  |  1. "광역도시계획"이란       ||
|  |                              |  |     2 이상의 특별시...        ||
|  |  도시 및 주거환경정비법       |  |                              ||
|  |  법률 제19XXX호 | 시행 2025. |  |  2. "도시·군계획"이란        ||
|  |                              |  |     특별시...                 ||
|  |  도시공원 및 녹지 등에 관한  |  |                              ||
|  |  법률                        |  |  3. "도시·군기본계획"이란    ||

|  |                              |  |                              ||
|  |  ...                         |  |  +------------------------+  ||
|  |                              |  |  |  [문서에 삽입]          |  ||

|  +------------------------------+  |  +------------------------+  ||

+------------------------------------------------------------------+
```

### 사용 컴포넌트

| 영역 | Ant Design 컴포넌트 | 설명 |
|------|---------------------|------|
| 검색 유형 | `Radio.Group` | 법령명/조문번호/키워드 전환 |
| 검색 입력 | `Input.Search` (size="large") | 엔터 또는 버튼으로 검색 |
| 고급 검색 | `Collapse` (Panel 1개) | 접힘/펼침 토글 |
| 법령 구분 | `Checkbox.Group` | 법률/시행령 등 필터 |
| 시행일 범위 | `DatePicker.RangePicker` | 시행일 필터 |
| 소관부처 | `Select` | 부처 선택 드롭다운 |
| 검색 결과 | `List` (pagination 포함) | 법령 목록 |
| 결과 항목 | `List.Item` + `Card` | 법령명, 번호, 시행일 표시 |
| 조문 상세 | `Card` + `Typography` | 조문 내용 표시 |
| 조문 탐색 | `Anchor` 또는 `Tree` | 조문 목차 네비게이션 |
| 문서 삽입 | `Button` (type="primary") | 현재 편집 중인 문서에 삽입 |
| 클립보드 복사 | `Button` + `message.success` | 복사 완료 알림 |
| 레이아웃 분할 | `Row` + `Col` (span 10 + 14) | 좌우 분할 |

### 상호작용

- 검색 결과 항목 클릭 시 우측 패널에 조문 상세 표시
- "문서에 삽입" 클릭 시:
  - 편집기가 열린 상태면 커서 위치에 조문 텍스트 삽입
  - 편집기가 닫힌 상태면 `Modal`로 대상 문서 선택 후 삽입
- 검색 시 `Spin` 로딩 표시
- 검색어 자동완성 (`AutoComplete`) 지원 (인기 검색어 기반)
- 법률 API 호출 실패 시 `Alert` 컴포넌트로 오류 메시지 표시
- 조문 상세에서 텍스트 드래그 선택 후 "선택 영역만 삽입" 가능

## 공통 UX 패턴

### 로딩 상태
- 페이지 전환: `Spin` (전체 콘텐츠 영역)
- 데이터 로드: `Skeleton` (테이블/카드 형태)
- 버튼 액션: `Button`의 `loading` 속성

### 알림/피드백
- 성공: `message.success` (상단 알림)
- 경고: `message.warning`
- 에러: `notification.error` (상세 내용 포함)
- 확인 필요: `Modal.confirm`

### 빈 상태
- 데이터 없음: `Empty` 컴포넌트
- 첫 사용: 가이드 카드 (`Card` + 설명 + CTA 버튼)
