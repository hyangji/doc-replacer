# DocReplacer 기능 목록

## 개요
도시계획 제안서 문서의 반복적 수정을 자동화하고 법률 문서 정확성을 검증하는 웹 시스템.

**배포 URL**: https://doc-replacer.vercel.app

---

## 1단계 (MVP) - 완료

### 1.1 문서 업로드 및 관리
| 기능 | 설명 | 상태 |
|------|------|------|
| 파일 업로드 | HWP, HWPX, Excel, Word 드래그앤드롭 업로드 (최대 50MB) | ✅ |
| 문서 목록 | 전체 문서 테이블 (페이지네이션 20건/페이지) | ✅ |
| 문서 삭제 | 개별 문서 삭제 | ✅ |
| HWPX 텍스트 추출 | ZIP+XML 구조에서 텍스트/테이블 파싱 | ✅ |
| 파일 DB 저장 | PostgreSQL 바이너리 저장 (로컬 파일시스템 불필요) | ✅ |

### 1.2 문서 편집기
| 기능 | 설명 | 상태 |
|------|------|------|
| Monaco Editor | VS Code 기반 텍스트 편집기 | ✅ |
| 검색 (Ctrl+F) | 텍스트 검색, 대소문자 구분, 정규식 지원 | ✅ |
| 치환 (Ctrl+H) | 개별 바꾸기 / 모두 바꾸기 | ✅ |
| 검색 하이라이트 | 검색 결과 노란색 강조, 현재 항목 주황색 | ✅ |
| 저장 (Ctrl+S) | 문서 내용 저장 + 버전 생성 | ✅ |
| 되돌리기/다시하기 | Ctrl+Z / Ctrl+Y | ✅ |
| 상태바 | 커서 위치, 총 줄수, 문자수, 검색 결과 수 | ✅ |

### 1.3 엑셀 일괄 교체
| 기능 | 설명 | 상태 |
|------|------|------|
| 엑셀 업로드 | .xlsx/.xls 파일 업로드 | ✅ |
| 자동 헤더 인식 | 다양한 헤더명 자동 매핑 (원본/수정, 찾기/바꾸기 등) | ✅ |
| 일괄 교체 실행 | 엑셀 규칙에 따라 문서 내용 일괄 치환 | ✅ |
| 교체 결과 표시 | 성공 건수 / 일치 항목 없음 알림 | ✅ |

### 1.4 Diff 비교
| 기능 | 설명 | 상태 |
|------|------|------|
| 비교 모드 | 원본(좌) vs 수정본(우) 분할 뷰, 단어 수준 diff | ✅ |
| 편집 모드 | 수정본 직접 편집 가능 (contentEditable) | ✅ |
| 줄별 하이라이트 | 수정(초록), 삭제(빨강), 되돌림(파랑) 색상 구분 | ✅ |
| 변경사항 요약 테이블 | 위치, 유형, 이전값, 새값 테이블 | ✅ |
| 테이블 row 클릭 포커스 | row 클릭 시 해당 변경 위치로 자동 스크롤 | ✅ |
| 원본으로 되돌리기 | 줄 단위 원본 복원 버튼 | ✅ |
| 수정본으로 되돌리기 | 되돌린 줄을 다시 수정본으로 복원 (토글) | ✅ |
| 모두 되돌리기 | 전체 변경사항 초기화 | ✅ |
| 이전/다음 네비게이션 | 변경사항 간 이동 + 스크롤 연동 | ✅ |
| 변경 보고서 다운로드 | 변경 내역 텍스트 파일 다운로드 | ✅ |
| 저장 | 수정된 내용 저장 API 호출 | ✅ |
| 스크롤 동기화 | 편집 모드에서 원본/수정본 스크롤 동기화 | ✅ |

### 1.5 버전 관리
| 기능 | 설명 | 상태 |
|------|------|------|
| 자동 버전 생성 | 저장/교체 시 버전 번호 자동 증가 | ✅ |
| 버전 히스토리 | 버전 목록 조회 | ✅ |
| 버전 되돌리기 | 이전 버전으로 문서 복구 | ✅ |

## UI/UX

### 페이지 구조
| 페이지 | 경로 | 설명 |
|--------|------|------|
| 대시보드 | `/` | 통계 카드, 퀵 액션, 최근 문서 |
| 문서 목록 | `/documents` | 전체 문서 목록 + 업로드 |
| 문서 편집기 | `/editor/[id]` | 탭 기반 (편집 / 엑셀 교체 / Diff 비교) |
| 법률 검색 | `/law` | 검색 + 상세 분할 뷰 |
| 맞춤법 검사 | `/spellcheck` | 입력 + 결과 분할 뷰 |

### 공통 컴포넌트
- 헤더 네비게이션
- 사이드바 메뉴
- 페이지 헤더 (제목 + 브레드크럼)
- 로딩 스피너

## 백엔드 API 엔드포인트

### 문서 관리
```
POST   /api/documents/upload                      문서 업로드
GET    /api/documents/                            문서 목록
GET    /api/documents/{id}                        문서 상세
DELETE /api/documents/{id}                        문서 삭제
POST   /api/documents/{id}/save                   문서 저장
POST   /api/documents/{id}/revert                 버전 되돌리기
GET    /api/documents/{id}/versions               버전 목록
GET    /api/documents/{id}/diff                   Diff 비교
```

### 검색/교체
```
POST   /api/documents/{id}/search                 텍스트 검색
POST   /api/documents/{id}/replace-text           검색/치환
POST   /api/documents/{id}/replace                JSON 일괄 교체
POST   /api/documents/{id}/excel-upload           엑셀 일괄 교체
POST   /api/documents/{id}/replace/preview        교체 미리보기
```

### 법률/맞춤법
```
GET    /api/law/search                            법령 검색
GET    /api/law/{law_id}                          법령 상세
POST   /api/law/verify                            법률 인용 검증
POST   /api/spellcheck/                           맞춤법 검사
POST   /api/spellcheck/legal-terms                법률 용어 검사
```
