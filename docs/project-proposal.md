# 도시계획 제안서 문서 관리 시스템

## 프로젝트 개요

도시계획 제안서 작성 업무에서 발생하는 반복적인 문서 수정 작업을 자동화하고,  
법률 관련 문서의 정확성을 검증하는 웹 기반 문서 관리 시스템입니다.

---

## 도입 배경

| 현재 문제점 | 시스템 도입 효과 |
|------------|----------------|
| 금액/숫자 변경 시 수작업으로 하나씩 수정 | 엑셀 업로드 한 번으로 일괄 자동 수정 |
| 법률 용어 오타를 사람이 일일이 확인 | 법률 API 연동으로 자동 검증 |
| 수정 전/후 비교가 어려움 | 좌우 비교 화면으로 한눈에 확인 |
| 문서 버전 관리가 안 됨 | 자동 이력 관리 및 복원 기능 |

---

## 핵심 기능

### 1. 한글(HWPX) 파일 기반 문서 처리
- 한글 파일(.hwpx) 업로드, 읽기, 수정, 저장
- 문서 내 텍스트 및 표 데이터 자동 인식
- 서식(글꼴, 크기, 표 레이아웃) 보존

### 2. 엑셀 기반 일괄 수정
- 수정할 내용을 엑셀 파일(.xlsx)로 업로드
- 한글 파일 내 표의 데이터를 엑셀 내용으로 자동 교체
- 매핑 규칙 설정 (어떤 표의 어떤 항목을 교체할지 지정)

> **사용 예시**  
> 한글 파일: `의자 | 5,000원` → 엑셀 파일: `의자 | 6,000원`  
> 결과: 한글 파일이 `의자 | 6,000원`으로 자동 수정됨

### 3. 오타 검출
- 한국어 맞춤법 자동 검사
- 법률 전문 용어 오타 검출 (커스텀 법률 용어 사전 기반)
- 오타 위치 하이라이트 및 수정 제안

### 4. 수정 전/후 비교 (Diff View)
- 좌측: 원본 문서 / 우측: 수정된 문서
- 변경된 부분 색상 하이라이트 (추가: 초록, 삭제: 빨강)
- 항목별 수정 승인 / 거절 선택 가능
- 되돌리기 / 저장하기 버튼

### 5. 검색 및 치환
- IDE 스타일의 찾기/바꾸기 인터페이스
- 대소문자 구분, 정규식 지원
- 하나씩 바꾸기 / 모두 바꾸기 선택
- 검색 결과 개수 및 위치 표시

### 6. 법률 검증 및 검색
- **법률 자동 검증**: 문서 내 인용된 법률명/조문번호의 존재 여부 자동 확인
- **법률 검색 화면**: 별도 화면에서 법령 검색, 조문 조회, 연혁 확인
- **최신 여부 확인**: 인용된 법률의 개정/폐지 여부 알림
- 연동 API: 국가법령정보 Open API (법제처, 무료)

## 추가 기능

### 7. 파일 변환
- **HWPX → Word(.docx)** 변환 및 다운로드
- **HWPX → PDF** 변환 및 다운로드
- 변환 시 서식 최대한 보존

### 8. 한글 금액 자동 변환
- 숫자 금액 변경 시 한글 표기 자동 갱신
- 예: `5,000,000원` 변경 → `일금 오백만원정` 자동 갱신

### 9. 금액 자동 계산
- 표 내 금액 변경 시 합계, 소계 자동 재계산
- 부가세(VAT) 자동 계산
- 계산식 오류 자동 감지

### 10. 버전 관리 (히스토리)
- 문서 수정 이력 자동 저장
- 특정 시점의 문서로 복원 가능
- 수정자, 수정 일시, 변경 내용 기록

### 11. 다중 파일 일괄 처리
- 여러 한글 파일을 동시에 업로드
- 하나의 엑셀 파일로 여러 문서를 한 번에 수정
- 일괄 처리 결과 요약 리포트

### 12. 변경 보고서 자동 생성
- 수정 완료 후 변경 내역 요약 보고서 자동 생성
- 변경 항목, 변경 전 값, 변경 후 값 목록
- 보고서를 PDF 또는 엑셀로 다운로드

### 13. PDF 내보내기
- 최종 완성된 문서를 PDF로 변환
- 인쇄용 고품질 PDF 생성
- 워터마크, 페이지 번호 등 옵션 설정

## 개발 단계

### 1단계 - 핵심 기능 (MVP)
- HWPX 파일 업로드/읽기/수정/저장
- 엑셀 기반 일괄 수정
- 검색 및 치환
- 수정 전/후 비교 (Diff View)

### 2단계 - 법률 연동
- 법률 오픈 API 연동
- 법률 검색 화면
- 법률 용어 오타 검출

### 3단계 - 고급 기능
- 파일 변환 (Word, PDF)
- 한글 금액 자동 변환
- 금액 자동 계산
- 일반 맞춤법 검사

### 4단계 - 운영 기능
- 버전 관리 (히스토리)
- 다중 파일 일괄 처리
- 변경 보고서 자동 생성
- PDF 내보내기
